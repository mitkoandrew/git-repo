Реализованы примеры рекурсивных алгоритмов:
1. Binary.h
Перевод в двоичную систему счисления
2. Exponentation.h
Возведение числа в степень
3. Calculator_3_to_20.h
подсчет кол-ва программ, преобразующих число 3 в 20 (при условии, что исполнитель делает две команды - увеличивает на 1 и умножает на 2)
4. Count_Route.h
Посчет кратчайших путей в матрице размера 3*3 из крайней левой верхней вершины в нижнюю
5. Chess_Move.h
Задача шахматного коня - поиск маршрута, прозодящего через все поля шахматной доски по олному разу
